< 통계 용어 > 
모집단 : 조사고자 하는 대상 집단 전체 (ex 대한민국의 20대)
모수 : 모집단의 평균 

표본집단 : 집단의 특성을 잘 나타낼 수 있는 일부 (ex 대한민국의 20대 중 서울에 사는)
통계량 : 표본집단의 평균 (이걸 통해 모수를 추정)

기술통계 : ex 평균을 낸다 등  
요약통계량 : 기술통계의 결과로 나온 값 = 평균값, 최대값, 최솟값, 사분위수 
추측통계 : 매년 100만원씩 일당이 늘었다. 내년에도 100만원 늘거다 -> 수식 f(x) = x+100

단위 : row , record
변수 : 각 단위에 대해 관측되는 특성, column이 변수 ex 성별(여남여여), 키(전체), 몸무게. 속성과 속성값 전부를 변수로 지칭함.  
데이터 : 하나 이상의변수에 대한 관찰값의 모음

확률화 : 모집단으로부터 편의되지 않는 표본을 추출하는 절차
확률 표본 : 확률화 절차에 의해 추출된 표본
표본오차 : 표본단위가 모집단을 대표하지 못해서 발생하는 오차 


통계분석의 2갈래

탐색적 데이터 분석 : 기술통계 / 객관적 / 데이터 요약
확증적 데이터 분석 : 추론통계 / 데이터 추론 

모형 : 수학 공식을 만들어내는걸 모형화 한다고 함
